What is Links OS?

Links OS is a custom non profit organization created operating system, built with Hyperlink Powered File Managment Engine.

Why Use Links OS?

Links OS is an advanced yet non demanding OS that can run on many platforms. Links OS is also in early development so YOUR FEEDBACK IS MUCH APPRECIATED!

What's new?

Links OS Version 3.2 just dropped with some Brand New Features including:

Bookmarks:

You can now add up to three bookmarks to your favorite sites, with custom names, and the ability to delete your bookmarks.

HTML editor:

You can now program your very own HTML program, which also includes a basic text editor.

Radio:

Links Radio station plays 24/7, currently there is only one station: Flute Channel. Be sure to tune in!

Files Managment System:

Links now has it's own file system, with this new update your OS comes preinstalled with Flute.mp3, and...

Links Graphing Calculator:

Comes with the new file management system, you can insert a slope, an intercept and bam! Watch Links make your graph for you!

What's Next?

Features to come soon: we are currently prioritizing bug fixes, and UI updates. Features that may or may not be added to the next Links update include more radio songs and/or stations, better file management system, games page, and more!
